AJAX Mapping

GET : 조회 		SELECT
POST: 생성		CREATE
PATCH: 수정		UPDATE
PUT : 대체(덮어쓰기)
DELETE: 삭제 		DELETE

-요청 URL 작성엔 경로명을 명사형으로 써주는 게 좋다
상품등록 		POST  	/items
상품목록 		GET    	/items
상품조회		GET    	/items/{id}   , /items?id=1
상품수정		PATCH 	/items{id}
상품삭제 		DELETE  	/items/{id}

REST API? 이런 방식을 그렇게 부른다는데.
이걸 사용해서 댓글/대댓글 기능 구현 가능.
